# 项目截图目录

将你的项目截图放在这个目录下。

## 命名规范

- `sonix-home.png` - SONIX首页截图
- `sonix-generate.png` - SONIX音乐生成页截图
- `sonix-admin.png` - SONIX管理后台截图
- `smart-park-1.png` - 智慧园区截图
- 等等...

## 推荐尺寸

- 宽度：1920px 或 1280px
- 高度：按比例
- 格式：PNG 或 JPG

## 如何截图

1. 打开你的项目
2. 按 F12 打开开发者工具
3. 点击设备模拟按钮，选择合适的分辨率
4. 使用截图工具截取

## 视频上传

演示视频建议上传到：
- B站（国内访问快）
- YouTube（国际化）

然后在 `app/projects/[id]/page.tsx` 中填入视频链接。
